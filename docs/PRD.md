# 📄 PRD —  Digital Lending Platform

## 1. Overview

A fully online lending platform where users can apply, get scored, and receive loan decisions without visiting a branch.

## 2. Target Users

* Salaried individuals
* Young professionals
* First-time borrowers

## 3. Core Features

* User authentication
* Loan application
* Automated credit scoring
* Loan approval/rejection
* Repayment schedule

## 4. User Flow

1. User registers
2. Completes profile
3. Applies for loan
4. System calculates score
5. Decision returned
6. Loan created
7. Repayment schedule generated

## 5. Functional Requirements

* Users can submit loan applications
* System calculates score instantly
* System auto-approves or rejects
* Admin can override decision

## 6. Non-Functional Requirements

* Fast response (<1s scoring)
* Secure authentication (JWT)
* Scalable backend

## 7. Success Metrics

* Application → decision time
* Approval rate
* User completion rate
